# Política de Privacidade – JulgaH2O

O aplicativo **JulgaH2O** foi desenvolvido com o objetivo de ajudar os usuários a manterem uma rotina saudável de hidratação por meio de lembretes, humor e interação divertida com o mascote JulgaFeliz.

Esta Política de Privacidade explica como coletamos, utilizamos e protegemos as informações do usuário.

---

## 1. Informações Coletadas

O app **NÃO coleta informações pessoais**, como nome, e-mail, telefone ou qualquer dado sensível.

O JulgaH2O coleta apenas:

### **1.1. Dados armazenados localmente no dispositivo**

- Quantidade de água ingerida por dia
- Histórico de hidratação
- Preferências do usuário (notificações, metas, mascote)

Esses dados ficam **somente no dispositivo** do usuário por meio do `AsyncStorage`.  
Nenhuma informação é enviada para servidores externos.

---

## 2. Uso das Informações

As informações coletadas são utilizadas apenas para:

- Calcular o progresso diário
- Exibir o mascote de acordo com o desempenho
- Mostrar o histórico no calendário
- Personalizar os lembretes
- Melhorar a experiência dentro do app

Nenhum dado é compartilhado, vendido ou enviado para terceiros.

---

## 3. Permissões Utilizadas

O JulgaH2O utiliza apenas a permissão:

### **Notificações**

Usada exclusivamente para enviar lembretes de hidratação.  
O usuário pode ativar ou desativar a qualquer momento.

---

## 4. Segurança

Como os dados ficam armazenados apenas no aparelho do usuário, adotamos medidas para garantir que:

- Nenhum dado pessoal é coletado
- Nenhuma informação é transmitida pela internet
- O usuário tem controle total sobre seus dados

---

## 5. Privacidade de Crianças

O aplicativo **não é destinado a crianças menores de 13 anos**, mas mesmo assim:

- Não coletamos dados pessoais
- Não mostramos anúncios direcionados para crianças

---

## 6. Alterações nesta Política

Podemos atualizar esta Política ocasionalmente.  
A data da última atualização sempre será exibida no topo do documento.
