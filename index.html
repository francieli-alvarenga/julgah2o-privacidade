Política de Privacidade - JulgaH2O

Esta Política de Privacidade descreve como o aplicativo JulgaH2O coleta, usa e protege suas informações.

1. Informações Coletadas

O app coleta apenas informações necessárias para funcionamento, como consumo de água, progresso diário e configurações do usuário.

2. Armazenamento Local

Todos os dados ficam armazenados apenas no seu dispositivo usando AsyncStorage. Nenhum dado pessoal é enviado para servidores externos.

3. Permissões

O app pode solicitar permissões para enviar notificações, usadas apenas para lembretes de hidratação.

4. Segurança

Seus dados nunca são compartilhados, vendidos ou transferidos.

5. Alterações

Esta política pode ser atualizada. A versão mais recente estará sempre nesta página.
